與 Markdown 發生衝突

　　有些狀況下，LaTeX 無法被正常渲染，如：

效果：

$$\lim { x\rightarrow \infty }{ \sum { k=1 }^{ x }{ \frac { \sin { k } +\cos { k } }{ k } } } $$

　　這個公式是完全正確的，但卻無法成功地被渲染，仔細一看才發現 { x\rightarrow \infty }{ \sum 是斜體顯示的。
　　Markdown中 _斜體_ 是斜體的語法，跳脫字元就能解決這個問題了。

效果：

limx→∞∑k=1xsink+coskk
